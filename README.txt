Introduction
============

This is a Plone 4 buildout (for development purposes only).

Installation
------------

To install::

    $ git clone git://github.com/aclark4life/Plone-4-Buildout.git
    $ cd Plone-4-Buildout
    $ python2.6 bootstrap.py -d
    $ bin/buildout
    $ bin/plone fg

Now check http://localhost:8080.

Have fun!
